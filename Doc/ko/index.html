GGUF Editor D++ へようこそ

GGUF Editor D++ は、GGUF および Safetensors 形式の言語モデル（LLM）ファイルの操作、分析、最適化のための最先端ツールです。Delphi で開発されており、研究者や開発者のための高性能なインターフェースを提供します。

メインインターフェース：テンソル可視化とモデル管理。

GGUF Editor D++ を使用する理由

  • マルチソース： GGUF モデルと PyTorch Safetensors の重みをマージできます。
  • リアルタイム可視化： インタラクティブなチャートを使用して、テンソルの分布と値を分析できます。
  • 高性能： 量子化・復量子化のために DLL（Llama.cpp）を最適化して使用します。
  • 高度な編集： マッピングを使用してメタデータ（KVs）やテンソル名を変更できます。

互換性

本ツールは、Q4_0, Q4_1, Q5_0, Q5_1, Q8_0, Q8_1, Q2_K, Q3_K, Q4_K, Q5_K, Q6_K, Q8_K, IQ4_NL, MXFP4, NVFP4 など、幅広い量子化タイプをサポートしています。
