クイックスタートガイド

1. マージのワークフロー

このガイドでは、ベースモデル（A）と重み（B）をマージして新しいモデルを作成する方法を説明します。

  1. 読み込み：
    • ベースとなる モデル A を Browse & Load します。
    • 特定の重みである モデル B を Browse & Load します。
  2. 選択：
    • 「Model B」タブで転送するテンソルにチェックを入れるか、「Transfer All」を使用します。
    • 層フィルターを使用して指定します（例：`16..32` や `Mod 2` で奇数層）。
  3. 確認（オプション）：
    • 「View Tensors」を開きます。
    • A と B を読み込み、「Diff (T1-T2)」タブを使用して値の差分を確認します。
  4. 出力：
    • 「Model Out」タブにはマージされたリストが表示されています。
    • 「出力」フィールドに出力パスを設定します。
    • Save をクリックします。

2. 可視化の使用

ブロック検出と外れ値を含む可視化の例。

ナビゲーション

  • 下部スライダー： 水平方向にビューをスクロールします。
  • マウスホイール： ズームイン/アウト。
  • クリック＆ドラッグ： ビューウィンドウをパン（移動）します。
  • キーボード： `+`/`-` でズーム、`<`/`>` でスクロール。

エクスポート

コンテキストメニューまたはエクスポートボタン（CSV/TXT形式）から、ヒストグラムまたは時系列データに表示されているデータをエクスポートできます。

3. 設定

設定 にアクセスして以下を行います：

  • 外部 DLL の使用を有効/無効にします（高速化）。
  • デフォルトのスプリットサイズを設定します。
  • 差分の増幅を設定します（最小限の偏差を確認するため）。
