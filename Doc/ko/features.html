詳細な機能

1. モデルの編集とマージ

本ツールは、最大3つのモデルソースを同時に読み込むことができます：

  • モデル A (GGUF)： ベースモデル。
  • モデル B (GGUF)： マージ用の2番目のモデル。
  • モデル S (Safetensors)： PyTorch 重みの統合。

特定のテンソルや層（すべての "blks"、または奇数/偶数層）を出力モデルに転送することができます。

2. 可視化と分析

「テンソル」ビューにアクセスして、モデル内部を検索します。

  • 時系列プロット： テンソル沿いの float 値を視覚化します。異常や外れ値を検出します。
  • 比較（T1 - T2）： 2つのテンソルを減算して差分を確認します（マージ分析に有用）。
  • ヒストグラム： 重みの分布（正規分布、ガウス分布など）を分析します。

3. 名前マッピング

マッピングエンジンにより、ターゲットアーキテクチャに合わせてテンソルを自動的にリネームできます。

  • 一般的なパターン（例：`blk.{}.attn.weight`）のサポート。
  • 外部構成ファイル（.txt）のサポート。
  • 無視するプレフィックスの管理。

4. インテリジェント量子化

テンソルタイプのリアルタイム変換：

  • Llama.cpp DLL： 高忠実度の復量子化のためにネイティブライブラリを使用（RMSE を最小化）。
  • Impl モード： DLL が利用できない場合や最大限の安定性が必要な場合の代替ソフトウェアアルゴリズム。
  • 量子化シミュレーション：実際に書き込む前に、FP32 テンソルが Q4_K に変換された場合にどうなるかを確認。

5. スプリットとマージ

統合されたツール：

  • スプリット： 大きな GGUF モデルを複数のファイルに分割（例：4GB ごと）。
  • マージ： 複数の GGUF シャードを1つの整合性のあるファイルに再結合。
