Funzionalità Dettagliate

1. Modifica e Fusione di Modelli

Lo strumento consente di caricare fino a tre fonti di modelli contemporaneamente:

  • Modello A (GGUF): Modello di base.
  • Modello B (GGUF): Secondo modello per la fusione (merging).
  • Modello S (Safetensors): Integrazione di pesi PyTorch.

Puoi trasferire tensori specifici o per layer (tutti i "blks", o layer dispari/pari) verso il modello di output.

2. Visualizzazione e Analisi

Accedi alla vista "Tensors" per ispezionare l'interno dei tuoi modelli.

  • Grafici Temporali (Serie): Visualizza i valori float lungo un tensore. Rileva anomalie e outlier.
  • Confronto (T1 - T2): Sottrai due tensori per vedere la differenza (utile per l'analisi di fusione).
  • Istogrammi: Analizza la distribuzione dei pesi (normale, gaussiana, ecc.).

3. Mapping dei Nomi

Il motore di mapping permette di rinominare automaticamente i tensori per adattare un modello a un'architettura target.

  • Supporto per pattern generici (es: `blk.{}.attn.weight`).
  • File di configurazione esterni (.txt).
  • Gestione dei prefissi ignorati.

4. Quantizzazione Intelligente

Conversione dei tipi di tensori in tempo reale:

  • DLL Llama.cpp: Utilizzo delle librerie native per una dequantizzazione ad alta fedeltà (RMSE minimizzato).
  • Modalità Impl: Algoritmo software alternativo per la massima stabilità o quando la DLL non è disponibile.
  • Simulazione di quantizzazione: Vedi come apparirebbe un tensore FP32 convertito in Q4_K senza scriverlo.

5. Split e Merge

Strumenti integrati per:

  • Split (Divisione): Dividere un modello GGUF grande in più file (es: 4GB).
  • Merge (Fusione): Ricombinare più shard GGUF in un unico file coerente.
