Architettura Tecnica

Documentazione destinata agli sviluppatori e ai contributori.

Tecnologie Utilizzate

  • Linguaggio: Delphi 10/11/12 (Object Pascal).
  • GUI: VCL (Visual Component Library).
  • Grafica: TeeChart (per grafici complessi).
  • FFI: Caricamento dinamico di DLL C/C++ (ggml-base.dll).

Struttura dei Moduli

File Unit Ruolo
uGGUFModel.pas Definizione delle classi dati (Tensori, KVs).
uGGUFReader.pas Lettura binaria dei file GGUF.
uGGUFWriter.pas Scrittura e manipolazione dei flussi di output.
uGgmlQuants.pas Interfaccia con le funzioni di quantizzazione (DLL o Impl).
uViewTensors.pas Logica di visualizzazione, rilevamento anomalie e grafici.
uTensorTranspose.pas Logica di trasposizione delle matrici (per Safetensors).

Formato del Mapping

I file di mapping sono file di testo (.txt) contenenti regole pattern=replacement.


model.layers.0.input_layernorm.weight -> attention.input_layernorm
model.layers.{}.feed_forward.w1.weight -> mlp.gate

Gestione della Memoria

L'applicazione utilizza un approccio "Lazy Load" per i dati dei tensori. I dati grezzi vengono decodificati in memoria (F32) solo nel momento della visualizzazione nei grafici o della quantizzazione, per minimizzare l'impatto sulla RAM su modelli molto grandi.
