Guía Rápida de Uso

1. Flujo de Trabajo de Fusión (Merge)

Esta guía explica cómo fusionar un modelo base (A) con pesos (B) para crear un nuevo modelo.

  1. Carga:
    • Navegar y Cargar Modelo A (Tu base).
    • Navegar y Cargar Modelo B (Tus pesos específicos).
  2. Selección:
    • En la pestaña "Model B", marca los tensores a transferir o usa "Transfer All".
    • Usa filtros por capas (ej. `16..32` o `Mod 2` para capas impares).
  3. Verificación (Opcional):
    • Abrir "View Tensors".
    • Cargar A y B. Usar la pestaña "Diff (T1-T2)" para verificar las diferencias de valores.
  4. Salida:
    • La pestaña "Model Out" contiene ahora la lista fusionada.
    • Define la ruta de salida en el campo "Output".
    • Haz clic en Save.

2. Uso de la Visualización

Ejemplo de visualización con detección de bloques y outliers.

Navegación

  • Deslizador inferior: Desplaza la vista horizontalmente.
  • Rueda del ratón: Zoom in/out.
  • Clic y arrastrar: Mover la ventana de vista (Pan).
  • Teclado: Usar `+`/`-` para zoom, `<`/`>` para desplazar.

Exportar

Puedes exportar los datos mostrados en el histograma o las series temporales mediante el menú contextual o los botones de exportar (formato CSV/TXT).

  • Ajustes del tamaño de split por defecto.
