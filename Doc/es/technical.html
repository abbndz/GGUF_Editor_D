Arquitectura Técnica

Documentación dirigida a desarrolladores y contribuyentes.

Tecnologías Utilizadas

  • Lenguaje: Delphi 10/11/12 (Object Pascal).
  • GUI: VCL (Visual Component Library).
  • Gráficos: TeeChart (para gráficos complejos).
  • FFI: Carga dinámica de DLLs C/C++ (ggml-base.dll).

Estructura de Módulos

Archivo Unit Rol
uGGUFModel.pas Definición de clases de datos (Tensors, KVs).
uGGUFReader.pas Lectura binaria de archivos GGUF.
uGGUFWriter.pas Escritura y manipulación de flujos de salida.
uGgmlQuants.pas Interfaz con funciones de cuantización (DLL o Impl).
uViewTensors.pas Lógica de visualización, detección de anomalías y gráficos.
uTensorTranspose.pas Lógica de transposición de matrices (para Safetensors).

Formato de Mapeo

Los archivos de mapeo son archivos de texto (.txt) que contienen reglas pattern=replacement.


model.layers.0.input_layernorm.weight -> attention.input_layernorm
model.layers.{}.feed_forward.w1.weight -> mlp.gate

Gestión de Memoria

La aplicación utiliza un enfoque de "Lazy Load" (carga diferida) para los datos de los tensores. Los datos sin procesar no se decodifican en memoria (F32) hasta el momento de la visualización en gráficos o la cuantización, con el fin de minimizar el consumo de RAM en modelos muy grandes.
