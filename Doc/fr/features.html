Fonctionnalités Détaillées

1. Édition et Fusion de Modèles

L'outil permet de charger jusqu'à trois sources de modèles simultanément :

  • Modèle A (GGUF) : Modèle de base.
  • Modèle B (GGUF) : Deuxième modèle pour fusion (merging).
  • Modèle S (Safetensors) : Intégration de poids PyTorch.

Vous pouvez transférer des tenseurs spécifiques ou par couches (tous les "blks", ou couches impaires/paires) vers le modèle de sortie.

2. Visualisation et Analyse

Accédez à la vue "Tensors" pour inspecter l'intérieur de vos modèles.

  • Graphiques Temporels (Séries) : Visualisez les valeurs de float le long d'un tenseur. Détectez les anomalies et les outliers.
  • Comparaison (T1 - T2) : Soustrayez deux tenseurs pour voir la différence (utile pour l'analyse de fusion).
  • Histogrammes : Analysez la distribution des poids (normal, gaussien, etc.).

3. Mapping de Noms

Le moteur de mapping permet de renommer automatiquement les tenseurs pour adapter un modèle à une architecture cible.

  • Support de patterns génériques (ex: `blk.{}.attn.weight`).
  • Fichiers de configuration externes (.txt).
  • Gestion des préfixes ignorés.

4. Quantification Intelligente

Conversion de types de tenseurs en temps réel :

  • DLL Llama.cpp : Utilisation des bibliothèques natives pour une déquantisation de haute fidélité (RMSE minimisé).
  • Mode Impl : Algorithme logiciel alternatif pour une stabilité maximale ou quand la DLL n'est pas disponible.
  • Simulation de quantification : Voir à quoi ressemblerait un tenseur FP32 converti en Q4_K sans l'écrire.

5. Split et Merge

Outils intégrés pour :

  • Split : Découper un gros modèle GGUF en plusieurs fichiers (ex: 4GB).
  • Merge : Recombiner plusieurs shards GGUF en un seul fichier cohérent.
