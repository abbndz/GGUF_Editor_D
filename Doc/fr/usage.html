Guide d'Utilisation Rapide

1. Workflow de Fusion (Merge)

Ce guide explique comment fusionner un modèle de base (A) avec des poids (B) pour créer un nouveau modèle.

  1. Chargement :
    • Browse & Load Model A (Votre base).
    • Browse & Load Model B (Vos poids spécifiques).
  2. Sélection :
    • Dans l'onglet "Model B", cochez les tenseurs à transférer ou utilisez "Transfer All".
    • Utilisez les filtres par couches (ex: `16..32` ou `Mod 2` pour les couches impaires).
  3. Vérification (Optionnel) :
    • Ouvrez "View Tensors".
    • Chargez A et B. Utilisez l'onglet "Diff (T1-T2)" pour vérifier les différences de valeurs.
  4. Sortie :
    • L'onglet "Model Out" contient maintenant la liste fusionnée.
    • Définissez le chemin de sortie dans le champ "Sortie".
    • Cliquez sur Save.

2. Utilisation de la Visualisation

Exemple de visualisation avec détection des blocs et outliers.

Navigation

  • Slider en bas : Fait défiler la vue horizontalement.
  • Molette de la souris : Zoom avant/arrière.
  • Clic-Glisser : Déplacer la fenêtre de vue (Pan).
  • Touches Clavier : Utiliser `+`/`-` pour zoomer, `<`/`>` pour glisser.

Export

Vous pouvez exporter les données affichées dans l'histogramme ou les séries temporelles via le menu contextuel ou les boutons d'export (format CSV/TXT).

3. Configuration

Accédez aux Paramètres pour :

  • Activer/ désactiver l'utilisation des DLL externes (plus rapide).
  • Réglages de la taille de split par défaut.
  • Amplification des différences (pour voir les écarts minimes).
