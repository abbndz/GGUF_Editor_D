Willkommen bei GGUF Editor D++

GGUF Editor D++ ist ein hochmodernes Werkzeug zur Bearbeitung, Analyse und Optimierung von Sprachmodellen (LLMs) im GGUF und Safetensors Format. Entwickelt in Delphi bietet es eine leistungsstarke Oberfläche für Forscher und Entwickler.

Die Hauptschnittstelle: Visualisierung von Tensoren und Modellverwaltung.

Warum GGUF Editor D++ verwenden?

  • Mehrfache Quellen: Verschmelzen Sie GGUF-Modelle mit PyTorch Safetensors-Gewichten.
  • Echtzeit-Visualisierung: Analysieren Sie Verteilungen und Werte von Tensoren mit interaktiven Diagrammen.
  • Hochleistung: Optimierte Nutzung von DLLs (Llama.cpp) für Quantisierung/Dequantisierung.
  • Erweiterte Bearbeitung: Ändern von Metadaten (KVs) und Tensor-Namen über Mapping.

Kompatibilität

Das Tool unterstützt eine breite Palette von Quantisierungstypen: Q4_0, Q4_1, Q5_0, Q5_1, Q8_0, Q8_1, Q2_K, Q3_K, Q4_K, Q5_K, Q6_K, Q8_K, IQ4_NL, MXFP4, NVFP4 und mehr.
