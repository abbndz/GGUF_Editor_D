詳細な機能

1. モデルの編集とマージ

最大3つのソースモデルを同時にロードできます：

  • モデル A (GGUF) : ベースモデル。
  • モデル B (GGUF) : マージ用の第2モデル。
  • モデル S (Safetensors) : PyTorch 重みの統合。

特定のテンソルやレイヤー単位（すべての "blks"、または奇数/偶数レイヤー）を出力モデルに転送できます。

2. 可視化と分析

「テンソル」ビューにアクセスして、モデルの内部を検査します。

  • 時系列プロット： テンソル沿いの float 値を可視化します。異常値や外れ値を検出します。
  • 比較 (T1 - T2) : 2つのテンソルを減算して差を確認します（マージ解析に有用）。
  • ヒストグラム： 重みの分布（正規分布、ガウス分布など）を分析します。

3. 名前マッピング

マッピングエンジンにより、ターゲットアーキテクチャに合わせてテンソル名を自動的にリネームできます。

  • 一般的なパターンをサポート（例: `blk.{}.attn.weight`）。
  • 外部設定ファイル (.txt) のサポート。
  • 無視するプレフィックスの管理。

4. インテリジェントな量子化

テンソルタイプのリアルタイム変換：

  • Llama.cpp DLL : ネイティブライブラリを使用して高忠実度の非量子化を行います（RMSE を最小化）。
  • Impl モード : DLL が利用できない場合や、最大安定性を求めるための代替ソフトウェアアルゴリズム。
  • 量子化シミュレーション：実際に書き出さなくても、FP32 テンソルが Q4_K に変換された際の状態を確認できます。

5. スプリットとマージ

統合されたツールによる操作：

  • スプリット (Split) : 大きな GGUF モデルを複数のファイルに分割します（例：4GBずつ）。
  • マージ (Merge) : 複数の GGUF シャードを1つの整合性の取れたファイルに再結合します。
