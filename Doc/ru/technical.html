Техническая архитектура

Документация для разработчиков и контрибьюторов.

Используемые технологии

  • Язык: Delphi 10/11/12 (Object Pascal).
  • GUI: VCL (Visual Component Library).
  • Графика: TeeChart (для сложных графиков).
  • FFI: Динамическая загрузка DLL C/C++ (ggml-base.dll).

Структура модулей

Файл Unit Роль
uGGUFModel.pas Определение классов данных (Тензоры, KVs).
uGGUFReader.pas Бинарное чтение файлов GGUF.
uGGUFWriter.pas Запись и манипуляция потоками вывода.
uGgmlQuants.pas Интерфейс с функциями квантования (DLL или Impl).
uViewTensors.pas Логика визуализации, обнаружения аномалий и графиков.
uTensorTranspose.pas Логика транспонирования матриц (для Safetensors).

Формат маппинга

Файлы маппинга — это текстовые файлы (.txt), содержащие правила pattern=replacement.


model.layers.0.input_layernorm.weight -> attention.input_layernorm
model.layers.{}.feed_forward.w1.weight -> mlp.gate

Управление памятью

Приложение использует подход "Lazy Load" для данных тензоров. Сырые данные декодируются в память (F32) только в момент отображения в графиках или квантования, чтобы минимизировать потребление RAM при работе с очень большими моделями.
