Краткое руководство по использованию

1. Рабочий процесс объединения (Merge)

Это руководство объясняет, как объединить базовую модель (A) с весами (B) для создания новой модели.

  1. Загрузка:
    • Выберите и загрузите Модель A (ваша база).
    • Выберите и загрузите Модель B (ваши специфические веса).
  2. Выбор:
    • На вкладке "Модель B" отметьте тензоры для передачи или используйте "Перенести все" (Transfer All).
    • Используйте фильтры по слоям (например, `16..32` или `Mod 2` для нечетных слоев).
  3. Проверка (Опционально):
    • Откройте "View Tensors".
    • Загрузите A и B. Используйте вкладку "Diff (T1-T2)" для проверки разницы значений.
  4. Результат:
    • Вкладка "Model Out" теперь содержит объединенный список.
    • Установите путь сохранения в поле "Output".
    • Нажмите Save.

2. Использование визуализации

Пример визуализации с обнаружением блоков и выбросов.

Навигация

  • Слайдер внизу: Прокручивает вид по горизонтали.
  • Колесо мыши: Масштабирование вперед/назад.
  • Клик и перетаскивание: Перемещение окна просмотра (Pan).
  • Клавиатура: Используйте `+`/`-` для масштаба, `<`/`>` для прокрутки.

Экспорт

Вы можете экспортировать данные, отображенные в гистограмме или временных рядах, через контекстное меню или кнопки экспорта (формат CSV/TXT).

3. Конфигурация

Доступ к Настройкам для:

  • Включения/отключения использования внешних DLL (быстрее).
  • Настроек размера фрагментации по умолчанию.
  • Усиления разницы (для просмотра минимальных отклонений).
